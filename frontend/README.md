# SKHU Track Frontend

`frontend/`는 SKHU Track의 사용자 웹 화면 디렉토리입니다.

## 개요

- Vanilla JavaScript 기반 MPA 구성
- 서버 응답 결과 화면 렌더링

## 주요 페이지

- `/`: 로그인 및 진입 화면
- `/grad/`: 대시보드
- `/profile/`: 프로필 설정
- `/grad/courses/`: 수강내역 관리
- `/grad/status/`: 졸업 판정 상세 확인
- `/storage/`: 자료함
- `/error/`: 공통 에러 페이지

## 기술 스택

- Vanilla JavaScript
- Vite
- Nginx
